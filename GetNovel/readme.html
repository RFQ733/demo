项目名

GetNovels

版本

3.1

项目：

  Python爬虫开发——从书趣阁中爬取小说《我的师兄太稳健了》的最新十章内并下载到此文件的子目录novels中，

使用说明

   请双击novel.py运行程序，运行后下载完的小说就到了novels文件夹中了。
   下载完小说后可以点击index.html中进行查看阅读
